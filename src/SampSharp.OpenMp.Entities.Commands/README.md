# SampSharp.OpenMp.Entities.Commands

SampSharp.OpenMp.Entities.Commands provides a command system for SampSharp open.mp gamemodes built on the entity-component framework. It allows you to define player commands using attributes on methods, with automatic parameter parsing and routing from in-game chat input.

## About SampSharp

SampSharp is a toolkit that allows you to write gamemodes in C# for open.mp servers using the .NET runtime. SampSharp's aim is to provide all features of object-oriented programming and modern .NET development.

For general documentation and guides, see [https://sampsharp.net](https://sampsharp.net).
